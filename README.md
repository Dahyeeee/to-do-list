<h1>Creating my own To-do-list by myself.</h1>

## 구현할 기능

- 할 일 입력하고 목록에 추가하는 기능
- 추가한 일 수정하는 기능
- 수정 중에 화면 아무곳이나 클릭하면 input box 없어지고 다시 text로 보이게 하는 기능
- 한 일 텍스트 클릭하면 cross out되는 기능
- 삭제 버튼 만들어서 누르면 목록에서 없어지는 기능
- 다 한 일만 없애는 기능
- 전부 삭제하는 기능
- 중요한 일 위로 올리는 기능
- 로컬저장소에 저장해서 refresh해도 남아있기

## 사용한 스펙

- react.js
- react-redux
- redux-toolkit
- css
